Release 0.3.0

March 7, 2017 Jamie

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Email

Info

After several months hard work there is virtually no visible change :) But it’s not all bad, most of the work went into code that supports more advanced features and making the existing features work properly so next release should have some more, different pretty pictures. The main visible difference will be in the default texture filtering which is now anisotropic and properly calculated from the ray differentials.

Multiple importance sampling for direct lighting is now also supported and used by default which leads to less noisy highlights on glossy surfaces.

New Features

  • Multiple Importance Sampling
  • Anisotropic texture filtering
  • Procedural nodes - well, you can now load WFObj (again) and VNF files.
  • Better light sampling
  • Geometry Instancing
  • Per-face shaders in PolyMesh

Images

Roadmap - Due May 2017

The next stage will include:

  • At least one exporter for a 3D package (probably Maya)
  • Transparency will be working again
  • Subdivision surfaces
  • Better error handling and reporting
  • Many general bugfixes and making everything more robust
  • Test scene suite and some more advanced demo scenes (the exporter will help here)
