Features

  • Distribution ray tracing
  • Disk light sources
  • Polygon meshes (+ WF .obj loading)
  • Basic texture support
  • BSDF support, Oren-Nayar diffuse and GGX specular
  • Transparency and thin-transparency support

Tech

Vermeer Light Tools is a physically plausible renderer.

Vermeer is written in Go so you’ll need that before building. Binaries will be made available when I’m ready for release.

Get Vermeer

go get github.com/jamiec7919/vermeer/cmd/vermeer
